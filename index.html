Industrial UI Library

Welcome to Industrial UI Library

A lightweight modular UI industrial & IOT component library:

✅ Built using TypeScript & Web Components

✅ Lightweight, no external dependencies other than dev-dependencies

✅ Global styling and variables allow defining a consistent yet customizable look & feel

✅ Tested using vitest

✅ Each component is well documented

✅ Demo showcase (this page)

Button

A customizable button web component with multiple style variants.

Usage:

<ind-button>Click Me</ind-button>

Attributes:

Attribute Type Default Description
disabled boolean false Disables the button
icon string undefined Specify the icon name. One of: plus, minus, close, check, etc
variant string "primary" Button style variant. One of: primary, secondary, success, info, warn, danger

Public API:

isDisabled(): boolean {...}
setDisabled(value: boolean): void {...}

Events:

A CustomEvent of type 'button-click-event' is triggered when the native event 'click' is called

Examples:

Basic Button
Button 
<ind-button>Button</ind-button>
Disabled State
Button 
<ind-button disabled="true">Button</ind-button>
With Icons
Add Download Search Trash 
<ind-button icon="plus">Add</ind-button>
<ind-button icon="download">Download</ind-button>
<ind-button icon="search" variant="success">Search</ind-button>
<ind-button icon="trash" variant="danger">Trash</ind-button>

Card

A customizable card web component.

Usage:

<ind-card>Contents</ind-card>

Attributes:

Attribute Type Default Description
header slot undefined Define the header of the card
footer slot undefiend Define the footer of the card

Example:

<ind-card>
    <div slot="header">Card Header</div>
    <p>Some card contents to showcase the card usage</p>
    <div slot="footer">
        <ind-button variant="primary">Close</ind-button>
    </div>
</ind-card>

Knob

A customizable knob web component.

Usage:

<ind-knob></ind-knob>

Attributes:

Attribute Type Default Description
disabled boolean false Disables the knob
grid boolean false Whether to display the reference grid or not
min number 0 Specify the minimum value
max number 100 Specify the maximum value
size number 140 Specify the width/height of the object (px)
symbol string % Specify the symbol to display after the value
value number 0 Specify the current value

Public API:

isDisabled(): boolean {...}
setDisabled(value: boolean): void {...}

Events:

A CustomEvent of type 'knob-change-event' is triggered when the knob changes its value

Examples:

Basic Knob
<ind-knob></ind-knob>
Default Value
<ind-knob value="50"></ind-knob>
Disabled State
<ind-knob value="33" disabled="true"></ind-knob>
Reference Grid
<ind-knob grid="true"></ind-knob>
Symbol Change
<ind-knob value="27" symbol="°"></ind-knob>
Min & max
<ind-knob grid="true" min="16" max="32" symbol="°"></ind-knob>

ToggleSwitch

A customizable toggle switch web component.

Usage:

<ind-toggle-switch></ind-toggle-switch>

Attributes:

Attribute Type Default Description
checked boolean false Whether the toggle switch is checked
disabled boolean false Disables the toggle switch
label string undefined Define the text label
variant string "primary" Toggle switch style variant. One of: primary, secondary, success, info, warn, danger

Public API:

isChecked(): boolean {...}
isDisabled(): boolean {...}
setChecked(value: boolean): void {...}
setDisabled(value: boolean): void {...}

Events:

A CustomEvent of type 'toggle-switch-toggled' is triggered when the native event 'change' is called

Examples:

Basic Toggle
<ind-toggle-switch></ind-toggle-switch>
Basic Toggle Checked
<ind-toggle-switch checked="true"></ind-toggle-switch>
Disabled State
<ind-toggle-switch checked="true" disabled="true"></ind-toggle-switch>
With Label
<ind-toggle-switch label="Toggle me!"></ind-toggle-switch>
Toggle Programmatically
