Industrial UI Library

Welcome to Industrial UI Library

A lightweight modular UI industrial & IOT component library:

✅ Built using TypeScript & Web Components

✅ Lightweight, no external dependencies other than dev-dependencies

✅ Global styling and variables allow defining a consistent yet customizable look & feel

✅ Tested using vitest

✅ Each component is well documented

✅ Demo showcase (this page)

// TODO: npm install industrial-ui

Button

A customizable button web component with multiple style variants.

Usage:

<ind-button>Click Me</ind-button>

Attributes:

Attribute Type Default Description
variant string "primary" Button style variant. One of: primary, secondary, success, info, warn, danger
disabled boolean false Disables the button

Example:

<ind-button variant="primary">Primary</ind-button>
<ind-button variant="secondary">Secondary</ind-button>
<ind-button variant="success">Success</ind-button>
<ind-button variant="info">Info</ind-button>
<ind-button variant="warn">Warn</ind-button>
<ind-button variant="danger">Danger</ind-button>
Primary Secondary Success Info Warn Danger

Card

A customizable card web component.

Usage:

<ind-card>Contents</ind-card>

Attributes:

Attribute Type Default Description
header slot undefined Define the header of the card
footer slot undefiend Define the footer of the card

Example:

<ind-card>
    <div slot="header">Card Header</div>
    <p>Some card contents to showcase the card usage</p>
    <div slot="footer">
        <ind-button variant="primary">Close</ind-button>
    </div>
</ind-card>

ToggleSwitch

A customizable toggle switch web component.

Usage:

<ind-toggle-switch></ind-toggle-switch>

Attributes:

Attribute Type Default Description
checked boolean false Whether the toggle switch is checked
disabled boolean false Disables the toggle switch
label string undefined Define the text label
variant string "primary" Toggle switch style variant. One of: primary, secondary, success, info, warn, danger

Example:

<ind-toggle-switch checked="true"></ind-toggle-switch>
<ind-toggle-switch checked="true" variant="secondary"></ind-toggle-switch>
<ind-toggle-switch checked="true" variant="success"></ind-toggle-switch>
<ind-toggle-switch checked="true" variant="info"></ind-toggle-switch>
<ind-toggle-switch checked="true" variant="warn"></ind-toggle-switch>
<ind-toggle-switch checked="true" variant="danger"></ind-toggle-switch>

<ind-toggle-switch checked="true" label="Label"></ind-toggle-switch>
